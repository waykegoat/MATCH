# 🎮 GamerMatch Bot

Telegram бот для знакомств геймеров и поиска напарников для игр.

## 🌟 Функции

- 📝 Создание анкет с играми и интересами
- 🔍 Поиск игроков по интересам или случайный поиск
- ❤️ Система лайков и мэтчей
- 💌 Взаимные лайки → обмен контактами
- 📸 Загрузка фото в анкету
- ⚙️ Настройки поиска
- 👨‍💼 Админ-панель со статистикой
